Тестовое задание (направление ПЛИС)

1. Имеется поток идентичных изображений 640х512 8bit, монохром - исходный файл изображения в архиве (alena_8bit.raw, 16-битный контейнер, bigEndian).
2. Поток выдается от источника внутри ПЛИС по AXIS строка за строкой, между строками имеется временной промежуток размером 100 системных тактов.
3. Каждая строка изображения передается по интерфейсу AXIS 8bit (с игнорированием сигнала ready приемника). 1 строка = 1 пакет AXIS.
4. Требуется разработать и верифицировать IP-ядро потоковой оконной фильтрации (фильтр Гаусса) изображения (без буферизации всего кадра). При заполнении горизонтальной рамки использовать зеркалирование.
5. Размер оконного фильтра 1x5, коэффициенты [0.028087, 0.23431, 0.475206, 0.23431, 0.028087] (преобразовать к фикс. точке).
6. На проверку выполнения задачи выносится ссылка на публичный github-репозиторий, в состав которого включается:
    - исходное изображение в формате .raw;
    - python-скрипт преобразования исходного .raw-изображения и отфильтрованного .raw-изображения в .png-файлы;
    - проект Vivado с исходным кодом ip-ядра и тестовым окружением;
    - README.md файл описания утилит и тестового окружения.

7. Разработку тестового окружения и кода IP-ядра выполнять в отдельных ветках. По завершению сформировать MR в master и влить ветки поочередно (не удаляя их).

Обзорная статья по оконным фильтрам - https://habr.com/ru/articles/142818/
